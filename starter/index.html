Show me the math

Keep the layers apart: the model as it is usually written, the manuscript's version, and the simplified engine the demo actually runs.

1 · The governing equation

The response relaxes toward the drive at a rate set by the damping:

dy/dt = u(t) - gamma*y(t)
2 · Steady state

Set the derivative to zero and the balance point falls out:

y_inf = u / gamma, tau = 1 / gamma
3 · With calibration constants · scaffolding is muted on purpose

Observation model. The grey symbols are calibration constants: they scale the result but carry no scientific meaning, so they recede.

S = S_0 * [ k_1*y + k_2*y^(2) ]
